## Clone the Repository

Open a terminal and run:

```bash
git clone https://github.com/USTH-classroom/ml-course.git
```

Then move into the course folder:

```bash
cd ml-course/docs/2-Linear-Regression
```

### Open the Notebook

Start Jupyter Notebook:
```bash
jupyter notebook
```

Then open: `01-data-mining-vs-machine-learning.ipynb`
